Project Overview

live narrative

This project is building a regenerative loop where food scraps become larvae feed, larvae become a resource, and the remainder becomes compost for soil health. Apprentices document the process, test variables, and help connect the system to the larger 3SUNS campus network.

Outputs

Inputs
